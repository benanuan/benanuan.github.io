# [初遇时的心动](https://benanuan.github.io/)

午后咖啡馆的玻璃窗上，雨水划出一道道不规则的纹路。他盯着那杯凉透的咖啡已经二十分钟了——自从她起身说去趟洗手间之后。

邻桌女孩的笑声尖锐地刺进耳朵，他突然意识到，自己竟在数她离开的分钟数。这不像他。三十岁的人，经历过几段不咸不淡的感情，早该对“心动”这个词免疫才是。

可当她推门进来的那一刻，头顶风铃叮当作响，雨后的阳光恰好穿过云层，在她发梢镀上一层朦胧的光晕。他突然理解了什么叫做“时间暂停”——不是文艺的夸张，而是真真切切地，全世界的声音都退去，只剩下自己鼓点般的心跳。

“抱歉，久等了。”她坐下时，袖口带起一阵淡淡的柑橘香。

后来他才知道，那是她护手霜的味道。

初次约会的话题总是谨慎而跳跃，像溪流中的石子，试探着深浅。他们聊工作、电影、最近读的书，唯独避开那些真正重要的问题——关于过往，关于期待，关于内心深处那些不敢轻易示人的角落。

送她到地铁口时，雨又下了起来。她没带伞，他说送她回家，她摇头说不用。两个人站在屋檐下，看雨丝在路灯下织成金色的网。

“下周，”她突然开口，“博物馆有个新展览，要不要一起？”

他说好，声音平静，心里却像被投入石子的湖面。

回去的路上，他反复回想她说话时微微上扬的嘴角，回忆她倾听时专注的眼神。到家后，他打开手机，点开对话框，输入又删除，最后还是只发了一句简单的“今天很开心，晚安”。

秒回的是一个笑脸。

他倒在床上，盯着天花板，突然笑了。原来心动从未消失，它只是躲在某个角落，等待合适的温度重新苏醒。
